🎯 Objectives
Explore flight price datasets to identify seasonal trends, regional differences, and airline pricing strategies.

Build interactive dashboards for visualizing KPIs such as average ticket price, route popularity, and demand fluctuations.

Apply data analysis techniques to support decision-making in travel and airline industries.

🛠️ Tools & Technologies
Power BI / Tableau – for interactive dashboards and visualizations.

Excel / CSV – raw flight price dataset.

Python (Pandas, NumPy, Matplotlib, Seaborn) – for data preprocessing and exploratory analysis.

DAX (Data Analysis Expressions) – for KPI calculations and advanced measures.

📂 Project Structure
flight_price.xlsx → Raw dataset containing flight details and prices.

scripts/ → Python scripts for cleaning and preprocessing data.

dashboard/ → Power BI or Tableau dashboards showcasing KPIs and insights.

README.md → Project documentation (this file).

📊 Key Insights
Seasonality: Prices vary significantly based on travel month and holiday seasons.

Airline Comparison: Certain airlines consistently offer lower fares on specific routes.

Route Popularity: High-demand routes show stronger price fluctuations.

🚀 How to Use
Clone the repository.

Open flight_price.xlsx in Power BI or Tableau.

Load the dataset and apply the provided DAX measures or Python scripts.

Explore dashboards to visualize KPIs and trends.
